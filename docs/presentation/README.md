# Presentation Materials

This directory contains presentation materials for the ASL&AI project.

## Files

- `ASL_AI_Presentation.pdf` - Main presentation (English, for portfolio)
- `ASL_AI_Presentation_ORT.pdf` - Original ORT presentation (Russian, reference)

## Adding Your Presentation

1. Translate your ORT presentation to English
2. Update accuracy numbers to 97.2% (if needed)
3. Export as PDF
4. Save as `ASL_AI_Presentation.pdf` in this directory
5. (Optional) Keep original as `ASL_AI_Presentation_ORT.pdf`

## Presentation Checklist

- [ ] Translated to English
- [ ] Updated accuracy to 97.2%
- [ ] Includes problem statement
- [ ] Shows system architecture
- [ ] Displays results and metrics
- [ ] Includes demo screenshots
- [ ] Mentions future work
- [ ] Professional formatting
- [ ] Saved as PDF
